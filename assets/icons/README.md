# Little Quest SVG Icon Set

목업 이미지 톤을 기준으로 제작한 일관된 SVG 아이콘 세트입니다.

## Style

- Soft nature education style
- Rounded line icons
- Main stroke: `#2F6B3F`
- Soft green fill: `#EAF4DF`
- Accent yellow: `#F4C84A`

## Usage

개별 SVG를 Flutter asset 또는 Web SVG asset으로 사용할 수 있습니다.
색상 변경이 필요하면 SVG 내부의 `#2F6B3F`, `#EAF4DF`, `#F4C84A` 값을 디자인 토큰으로 치환하세요.
